# Student Grade System (C++)

Programa simples feito em C++ para cadastrar alunos e calcular médias.

## Funcionalidades 
- Cadastro de multiplos alunos
- Entrada de idade, sexo e notas
- Cálculo automático da média de cada aluno 
- Determinação de aprovação ou reprovação com base na média minima configurável
- Verificação de aprovação
- Estatísticas:
* Percentual de aprovados/reprovados de acordo com o sexo
* Média de idade dos alunos aprovados/reprovados
* Relatório final com status de cada aluno

## Tecnologias utilizadas
- Linguagem: C++
- IDE: Visual Studio Code
- Compilador: MinGW

## Como compilar
```bash
g++ prog01.cpp -o prog01
./prog01
```
